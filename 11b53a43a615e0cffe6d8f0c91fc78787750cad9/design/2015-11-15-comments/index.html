Now with Comments!

For a blog you need comments. Like all those other jekyll themes we baked in Disqus. It's easy to set, it works and makes a static jekyll blog more dynamic.

If you want to use Disqus-Comments with Feeling Responsive, just open config.yml and add your disqus_shortname. More on how to use Disqus ›

By default comments are turned off. You can customize the default behaviour in config.yml. To turn on comments just add comments: true to front matter using the page layout layout: page.
